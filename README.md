# Dashboard de Análise de Veículos

## Descrição

Este projeto é um aplicativo web interativo desenvolvido com Streamlit que permite analisar dados de anúncios de vendas de carros. O dashboard oferece visualizações interativas para explorar padrões nos dados de veículos usados.

## Funcionalidades

- **Visualização de Histograma**: Analise a distribuição da quilometragem dos veículos
- **Gráfico de Dispersão**: Explore a relação entre preço e quilometragem
- **Interface Interativa**: Botões clicáveis para gerar gráficos sob demanda
- **Dashboard Responsivo**: Interface amigável e fácil de usar

## Como Usar

1. **Acesse o aplicativo online**: [Link do seu app no Render]
2. **Clique nos botões** para gerar diferentes tipos de gráficos
3. **Explore os dados** através das visualizações interativas

## Tecnologias

- **Python**: Linguagem de programação
- **Streamlit**: Framework para aplicações web
- **Plotly Express**: Biblioteca para gráficos interativos
- **Pandas**: Manipulação e análise de dados
